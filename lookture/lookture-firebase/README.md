
# Lookture · React + Firebase (Starter)

## Requisitos
- Node 18+
- Cuenta Firebase

## Pasos rápidos
1) `cp .env.example .env.local` y pega tus credenciales de Firebase Web.
2) `npm i`
3) `npm run dev` → http://localhost:5173

## Deploy
- `npm run build`
- `npm i -g firebase-tools`
- `firebase login`
- `firebase init hosting` (elige `dist`)
- `firebase deploy`
